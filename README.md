# my-react-starter

A starter for React project

## Installation

Clone the repo and run `npm install`

## Start

After the successfull installation of the packages: `npm run dev`

## Stack

- Typescript
- Vite
- React
- Mobx
- Tailwind CSS
- Ant design react (load when needed)
